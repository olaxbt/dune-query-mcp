DuneLink

Welcome to DuneLink

A modern bridge connecting Dune Analytics data to intelligent agents.

Available Endpoints:

Health Check

GET /dune/health

Check if the service is running correctly.

Get Latest Query Results

GET /dune/query/{query_id}/latest

Retrieve the latest results for a specific Dune query.

Execute Query

POST /dune/query/{query_id}/execute

Execute a Dune query and return the results.

MCP Tools:

This server exposes the following MCP tools:

  • get_latest_result(query_id: int): Fetch the latest results of a Dune query
  • run_query(query_id: int): Execute a Dune query and retrieve results
